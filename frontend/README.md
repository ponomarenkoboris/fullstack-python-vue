# Проект для создания опросов [Frontend]

## Скачивание папки проекта из репозитория 
```
git clone https://github.com/ponomarenkoboris/fullstack-python-vue.git
```

## Установка зависимостей
```
yarn install
```
## Запуск на локальном сервере
```
yarn serve
```
## Создание папки dist
```
yarn build
```
